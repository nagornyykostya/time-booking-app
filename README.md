Date and time booking React app
Visit it: https://time-booking-react-app.netlify.app/
==================

Installing
Run npm install

Run
npm start
Open http://localhost:9007 in a web browser.
The tech stack is:
HTML5
CSS3
Flexbox
Sass (Scss)
BEM methodology
JS ES6
ReactJS
Redux
Redux-thunk
Moment.js
